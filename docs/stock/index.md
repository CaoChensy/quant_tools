> 示例

```python
from wisecon.stock.index import *

for index_name in ["沪深300", "上证50", "中证500", "科创50"]:
    print(index_name)
    data = IndexStock(index_name=index_name, size=5).load()
    print(data.to_frame(chinese_column=True).to_markdown())
    print()
```
